## ScreenPixel — Удобный скриншотер для Windows и macOS

### 1. Личный кабинет

**Описание:**
Разработка личного кабинета для управления счетом, выбор плана для работы с скриншотером.

### 2. Авторизация

**Описание:**
Авторизация по паролю и соц. сетям для работы с системой.

### 3. Главная страница

**Описание:**
Реализация главной страницы с описанием всех преимуществ проекта.
