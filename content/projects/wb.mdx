## WBInform — Платформа управления заказами Wildberries

### 1. Автоматизированная система уведомлений

**Описание:**
Отправка сообщений по статусам заказов + контроль дублирования уведомлений.

### 2. Конструктор сообщений

**Описание:**
Разработка удобного интерфейса, который представляет собой конструктор сообщения. Доступна мультиканальная отправка (SMS/WhatsApp/Telegram), возможность использовать шаблон с динамическими переменными.

### 3. Менеджер API-ключей

**Описание:**
Работа с API-ключами в личном кабинете. Удобный визуальный интерфейс и централизованное хранение ключей.

### 4. Аналитика обратной связи

**Описание:**
Парсер отзывов с сегментацией по оценкам + кастомные триггеры для ПВЗ.
